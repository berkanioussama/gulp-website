# gulp-website
Gulp 4 basic flow

# Before you start

## 1. Install node
    install NodeJs 10.x.x or hier
    https://nodejs.org

## 2. Install the gulp command line utility
    npm install --global gulp-cli

## 3. Install the gulp packages
    npm install

## 4. Make sure you have Gulp 4
    gulp --version
    CLI version 2.0.1
    Local version 4.0.0

# How to start
In website directory Run commend

    gulp
